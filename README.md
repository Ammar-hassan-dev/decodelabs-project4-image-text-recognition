# 👁️ Image Text Recognition (OCR)

An AI-powered Optical Character Recognition (OCR) web application that extracts text from images using OpenCV and Tesseract OCR.

## ✨ Features

- Image Upload
- Text Extraction
- OCR Processing
- Confidence Validation
- Image Preprocessing

## 🛠️ Technologies

- Python
- Flask
- OpenCV
- Tesseract OCR
- HTML
- CSS
- JavaScript
