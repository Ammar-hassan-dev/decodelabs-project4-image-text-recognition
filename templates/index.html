HOW THE 4-STEP PIPELINE WORKS
1

INGESTION

Capture raw image bytes — an upload or a bundled sample — as a 3D pixel array (H × W × RGB).

2

PRE-PROCESSING

Grayscale collapses color depth, Gaussian Blur removes noise, Otsu's method forces a binary black/white decision per pixel.

3

RECOGNITION

pytesseract (Tesseract's CNN + Bi-LSTM engine) reads the cleaned image using the selected PSM layout mode.

4

VALIDATION

The 80% Gate: if confidence >= 0.80: keep() else: drop() — only validated words reach the output.
